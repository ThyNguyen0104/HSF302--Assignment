Hướng dẫn

  • Kéo thả ảnh vào vùng upload hoặc nhấn Chọn File.
  • Hỗ trợ JPG, PNG, WEBP, GIF, AVIF. Tối đa 10MB.
  • Chọn thư mục đích để phân loại ảnh trước khi upload.
  • Upload lên Cloudinary nếu đã cấu hình API key trong application.properties.
  • Chưa có API key? Ảnh sẽ lưu local server và có thể truy cập ngay.
  • Sau upload, copy URL và dán vào form thêm hoa để gán ảnh cho sản phẩm.
